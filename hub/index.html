Programme Dashboard — Part of the Gemba Suite

A read-only dashboard that aggregates Gemba Suite JSON exports into a unified programme view. Load A3 exports to see portfolio status, overdue actions, and programme health at a glance.

No data is stored — this is a stateless viewer. Files are processed in-browser and discarded when you close the page. Use ↓ Save to export a programme snapshot, and ↑ Open to reload it next session.

Get Started

Get Started

📱

Load an app on your phone or tablet

Every tool runs in your browser — no app store needed. For the best experience, install it on your home screen so it works like a native app, including offline.

Android — Open the tool in Chrome. Tap the install banner, or use the three-dot menu and select "Install app" or "Add to Home Screen".
iPhone / iPad — Open in Safari (not Chrome). Tap the Share button, then "Add to Home Screen".
Desktop — Open in Chrome or Edge. Click the install icon in the address bar, or just bookmark it.
💾

How your data is stored

Your data is stored locally in your browser using localStorage. It stays on your device — never sent to a server, never uploaded to the cloud, never shared.

Data persists between sessions — close the browser and come back, your work is still there.
Data is specific to the browser you used. Switch browsers or devices and your data won't follow — use JSON export to move it.
Clearing your browser data (cache, cookies, site data) will permanently delete your saved work.
📦

Back up and export your data

Every app has a JSON export button. It creates a file you can save, share, or reload later. Make exporting a habit — it's your backup and your way to share work with colleagues.

JSON export saves everything — all data, settings, and photos in one file. Use it after every session.
JSON import reloads a previously exported file. Use it to restore a backup or move data to a new device.
Excel / Print / SVG exports create outputs for analysis, presentations, and reports.
⚡

Working offline

All Gemba Suite tools work without an internet connection once loaded for the first time. Poor hospital WiFi is never a barrier to data collection at the gemba.

Install the app to your home screen for the most reliable offline experience.
Data entered offline is saved to your device automatically — no sync step needed.
The AI coaching export generates a prompt you paste later — it works whenever you next have connectivity.
No accounts. No IT department. No setup. Every tool is a single HTML file that runs entirely in your browser. There's nothing to install through IT, no passwords to manage, and no data processing agreements to sign. Open the link, start working. If your trust needs an IG review, the entire source code is in that one file — readable by anyone.

About

Built for the Gemba — where the real work is done.

Every tool in the Gemba Suite is a single HTML file that runs entirely in your browser. No accounts, no cloud storage, no IT department required. Your data stays on your device until you choose to export it.

The tools are designed for mobile-first use at the lab bench, the ward, or the clinic — wherever the work happens — the gemba. They work offline, so poor hospital WiFi is never a barrier to improvement.

Gemba Suite is a process improvement toolkit, not a clinical system. It does not connect to hospital systems, does not process patient data, and does not require regulatory approval. It is the digital equivalent of the whiteboards, sticky notes, and Excel spreadsheets that improvement teams already use — just faster, more structured, and available on any device.

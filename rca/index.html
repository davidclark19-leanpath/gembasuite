About Gemba-RCA

Gemba-RCA v1.2 is a structured root cause analysis tool for NHS healthcare teams implementing lean methodology.

Part of the Gemba Suite — free, single-file web apps for lean improvement at the gemba.

Information Governance

No patient-identifiable data fields. All data stays on this device in your browser's localStorage until you explicitly export it. No backend, no cloud storage, no tracking.
